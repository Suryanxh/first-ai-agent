# 🤖 First AI Agent

My first AI agent built while learning **LangChain** and **Agentic AI**.

It uses a **Groq LLM** with custom tools to solve simple mathematical queries and demonstrates how an AI agent reasons before producing an answer.

## Features

- Groq LLM integration
- Custom tools (Add, Multiply, Divide, Square Root)
- LangChain Agent
- Simple execution trace
- Beginner-friendly code

## Tech Stack

- Python
- LangChain
- Groq
- python-dotenv

## Example

```
User: What is 42 + 58?

→ Agent selects the Add tool
→ Tool returns: 100
→ Final Answer: 100
```
